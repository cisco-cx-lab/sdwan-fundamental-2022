CX SD-WAN Fundamental Training Lab Guide

Introduction

Welcome to CX SD-WAN fundamental training lab. This lab covers basic SD-WAN knowledge including onboarding WAN Edge, configuring feature device template, basic Hub-n-Spoke central control policy and common migration scenario.

The lab is intended for attendee has no prior SD-WAN knowledge. The instruction in this lab guide can be used for this training and also reference for future deployment.

Attendee for this training will gain SD-WAN knowledge and hands on experience from this lab.

1. Lab Access

The lab is designed using PoC Tool. To save attendee's time, the lab topology is pre-loaded and deployed. Each attendee has own pod with exact same setting to work on. To access the pod, attendee needs to anyconnect into dCloud using the provided login credential.
